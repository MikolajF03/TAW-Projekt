# TAW-Projekt
Aplikacja webowa opracowana w ramach przedmiotu „Technologie aplikacji webowych”, wykorzystująca Angular po stronie klienta oraz Node.js po stronie serwera.
___
## Uruchamianie
### backend

cd backend

npm install

npm start

### frontend

cd frontend

npm install

ng serve

___
## Lista zaimplementowanych funkcjonalności

- Edycja i usuwanie postów
- Paginacja postów
- Walidacja formularzy
- System polubień (like/unlike)
- API pogodowe
- Przełączanie motywu (jasny / ciemny – Dark Mode)
- System autoryzacji użytkowników:
  - Rejestracja i logowanie
  - Obsługa sesji / tokenów (np. JWT)
  - Zabezpieczone endpointy backendu
  - Ograniczony dostęp do wybranych funkcjonalności dla niezalogowanych użytkowników
